# Jury Hopps 🐰⚖️

**Better decisions through engineered disagreement.** File your case, and the rabbit Judge empanels four AI jurors — tool-using agents cast for maximal relevant disagreement — who research, argue, rebut, and deliver a verdict that preserves the dissent, exported as a shareable decision brief.

Built for the **Phoebe prize track** — AI to Coordinate the Real World.
